# swag-token
Polygon ERC-20 token test

### Setup Environment
```shell
npm install
```
### compile
```shell
npm run compile
```
### deploy(polygon mainnet)
```shell
npm run main:deploy
```
### deploy(polygon mumbai)
```shell
npm run test:deploy
```
### verify(polygon mainnet)
```shell
npm run main:verify
```
### verify(polygon mumbai)
```shell
npm run test:verify
```
### coverage
```shell
npm run coverage
```
### test
```shell
npm run test
```
